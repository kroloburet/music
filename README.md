## Strudel Music

Making music in [Strudel](https://strudel.cc) is fun. I created this repository to store the source code of my compositions created in Strudel. Select the composition in the directory `./strudel` and open the file `index.txt`. You can use the source code from this file in Strudel to play the composition.
